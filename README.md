# Prometheus on Koyeb

This repository contains the necessary files to deploy a Prometheus instance on Koyeb.

## Files
- `config/prometheus.yml`: Configuration file for Prometheus.
- `Dockerfile`: Dockerfile to build the Prometheus image.
